Comunicação Cliente x Servidor

História da Web

  1. Início da Criação da Internet

    2. Durante a Guerra Fria, foi necessário arrumar uma forma de proteger as informações e comunicações das pesquisas americanas dos soviéticos, por isso começaram as pesquisas e o desenvolvimento tecnológico com o objetivo de encontrar um meio de comunicação seguro e que sobrevivesse a um ataque nuclear.

    Em 1965 conseguiram realizar uma conexão bem sucedida de 2 computadores distantes com o conceito de uma rede galáctica para acessar rapidamente dados de qualquer lugar do mundo, mas o nascimento da Internet foi considerado ao criar a ARPANET, no qual, governo americano permitiu que as pesquisas fossem realizadas não só pelos militares como também pelas universidades.

    Em 1969, houve uma conexão realizada com sucesso entre a Universidade da Califórnia e a de Stanford e a primeira queda na Conexão, meses depois 4 universidades americanas estavam conectadas. E a partir daí, deu-se início ao uso e ao aprimoramento da Internet.

  2. Surgimento da Web

    3. Em 1989, Tim Berners-Lee criou a World Wide Web, ou seja, o tão famoso www que tanto utilizamos hoje! Este sistema foi criado para conectar as universidades com o intuito de fazer com que os trabalhos e as pesquisas acadêmicas fossem utilizados ao mesmo tempo. Este cientista também é responsável pelo desenvolvimento de duas ferramentas indispensáveis para a Internet: o Código HTML e o Protocolo HTTP.

    Muitas vezes, Internet e Web parecem ter o mesmo significado, mas não têm, elas tem características diferentes, pois a Internet é uma rede que conecta milhões de computadores pelo mundo, enquanto a Web é uma das ferramentas utilizadas para acessar a Internet.

  3. World Wide Web (www)

    4. Funciona como um Sistema de Sistribuição de Documentos de Hipertexto (HTTP) interconectados e acessíveis por meio de um Navegador Web conectado à Internet. Com o www, conseguimos acessar a Internet nesse formato que usamos hoje. Antes da Web e dos Navegador Web, a forma de acessar a Internet era por meio do Unix.

O que são Clientes?

Cliente é o Computador que está do lado do Usuário e que foi configurado para utilizar o Serviço que é oferecido pelo Servidor, ou seja, o Computador Cliente é aquele que se ocupa de um Serviço.

O que são Servidores?

Servidor é o Computador no qual foi configurado em uma Rede para oferecer algum tipo de Serviço, ou seja, o Computador Servidor é aquele que oferece um Serviço ao Usuário.

O que é HTML?

HTML (HyperText Markup Language ou "Linguagem de Marcação de Hipertexto") é uma Linguagem de Marcação utilizada na construção de Páginas na Web que é interpretada pelos Navegadores Web. A tecnologia é fruto da junção entre os padrões HyTime e SGML.
  • HyTime: É um padrão para a Representação Estruturada de Hipermídia e conteúdo baseado em Tempo, ou seja, é visto como um Conjunto de Eventos Concorrentes dependentes de Tempo (como áudio, vídeo, etc.), conectados por Hiperligações (em inglês: Hyperlink e Link);
  • SGML: É um padrão de formatação de textos que NÃO foi desenvolvido para Hipertexto, mas tornou-se conveniente para Transformar Documentos em Hiper-objetos e para Descrever as Ligações.

Introdução ao HTML na Prática

O que é e como funciona o Inspetor de Elementos?

  1. DevTools: São Conjuntos de Ferramentas de Desenvolvimento integradas aos Navegadores modernos (como Chrome, Firefox, Edge e Safari). Cada Navegador apresenta seu próprio Painel de DevTools, mas a ideia central é a mesma:

    1. Fornecer recursos para Inspecionar;
    2. Editar e/ou Analisar a Página sem que seja necessário sair do próprio Navegador.

  2. Importância do DevTools:

    1. Visualizar a Estrutura do DOMe suas Alterações em Tempo Real;
    2. Alterar Propriedades e Atributos HTML para Testar Comportamentos;
    3. Analisar Problemas e Debugar Possíveis Erros;
    4. Compreender o Funcionamento das Páginas;
    5. Realizar Ajustes Rápidos.

  3. Acesso ao DevTools:

    1. Browser Chrome (ou Edge Chromium): Clique com o botão direito em qualquer local da página e selecione “Inspecionar elemento” ou use o Atalho F12;
    2. Firefox: Clique com o botão direito e escolha “Inspecionar” ou use Ctrl + Shift + C (no Windows) / Cmd + Opt + C (no Mac). Navegando pelo Painel de Elementos.

  4. Benefícios da DevTools: A visualização da Aba Elementos (Chrome) ou do Inspector (Firefox) tem como vantagem:

    1. Mostrará a Árvore de Elementos do HTML (DOM) em Tempo Real;
    2. Identificará a Tag onde determinado Conteúdo ou Componente está localizado;
    3. Verificará quais são os Atributos utilizados (como id e/ou class)
    4. Mostrará a Estrutura Hierárquica e ficará mais fácil de entender como as Tags estão Aninhadas;
    5. O Código HTML pode ser editado diretamento do Navegador através da Aba Elementos..

Como é a Estrutura Básica do HTML?

A Estrutura Básica do HTML é composta por algumas Tags importantes, mas qual é a função de cada uma delas?
  1. <!DOCTYPE html> : Informa ao Navegador que esse Documento é do Tipo HTML e indica sua Versão;
  2. <html> : É a Raiz do Documento, serve com um Container que engloba TODOS os outros Elementos HTML;
  3. <body> : É onde fica todo o conteúdo de texto, imagem e vídeos, em que o usuário ou usuária vê e interage, nele os conteúdos são estruturados pelas demais tags do HTML;
  4. <head> : Compreende as Informações do Documento que serão Interpretadas pelo Navegador (Metadados), como, por exemplo, o <title>;
  5. <meta> : Define Metadados, ou seja, Informações sobre Dados de um Documento HTML, como, por exemplo, o Autor ou a Autora do Documento
  6. <link> : É uma tag vazia, que contém apenas atributos e faz a relação do documento HTML com recursos externos. É comumente usado para vincular uma folha de estilo externa, também é usada para definir o favicon da página (ícone da aba do Navegador), como outros recursos;
  7. <style> : Essa tag é usada para declarar estilos (CSS) para um documento.
